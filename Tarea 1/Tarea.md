# Glosario
1. **Firmware**: Es un tipo de software que esta integrado en un dispositivo hardware para controlar su funcionamiento basico.

2. **Datos**: Son hechos, cifras o simbolos en su forma basica sin procesar que pueden ser procesados y trasmitidos por un sistema informatico.

3. **Prodecimientos**: Son un conjunto de instrucciones o pasos organizados que se ejecutan para realizar una tarea especifica dentro de un programa.

4. **Usuarios**: Son las personas o entidades que interactúan con un sistema informático, software, aplicación o dispositivo para realizar tareas.

5. **Código embebido**: Se refiere a fragmentos de código que están integrados o insentados dentro de otro código o sistemas para cumplir una función.

6. **Contenedores**: Son una tecnología de virtualización a nivel de un sistema operativo que permite empaquetar una aplicación junto a sus dependencia.

7. **APIs**: Es un conjunto de definiciones, protocolos y herramientas que permiten que diferentes aplicaciones o sistemas se comuniquen entre si.

8. **Virtualización**: Es una tecnologia que permite crear una versión virtual de un recurso fisico, como un servidor, sistema operativo o red.

9. **Comunicaciones**: Se refieren al proceso mediante el cual dos o mas dispositivos, sistemas o programas intercambian información.

10. **Servidor**: Es un sitema informático diseñado para proporcionar servicios, recursos o datos a otros dispositivos o programas.

11. **CPU**: Es el componente principal de un ordenador o sistema informático encargado de interpretar y ejecutar las instrucciones de los programas.

12. **Cifrado**: Es un proceso mediante el cual la información o datos se transforman utilizando algoritmos matematicos.

13. **Utilitarios**: Son aplicaciones diseñadas para ayudar a gestionar, mantener, optimizar y proteger lel sistema operativo y los recursos.

14. **Nube**: Se refiere a la entrega de servicios informáticos a travez de internet, en lugar de hacerlo desde un servidor o computadora local.

15. **Ciberseguridad**: Es el contenido de prácticas, tecnologias y procesos diseñados para proteger sistemas informáticos, redes y dispositivos.
