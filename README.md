# Glosario
1. **Firmware**: Es un tipo de software que esta integrado en un dispositivo hardware para controlar su funcionamiento basico.

2. **Datos**: Son hechos, cifras o simbolos en su forma basica sin procesar que pueden ser procesados y trasmitidos por un sistema informatico.

3. **Prodecimientos**: Son un conjunto de instrucciones o pasos organizados que se ejecutan para realizar una tarea especifica dentro de un programa.

4. **Usuarios**: Son las personas o entidades que interactúan con un sistema informático, software, aplicación o dispositivo para realizar tareas.

5. **Código embebido**: Se refiere a fragmentos de código que están integrados o insentados dentro de otro código o sistemas para cumplir una función.

6. **Contenedores**: Son una tecnología de virtualización a nivel de un sistema operativo que permite empaquetar una aplicación junto a sus dependencia.

7. **APIs**: Es un conjunto de definiciones, protocolos y herramientas que permiten que diferentes aplicaciones o sistemas se comuniquen entre si.

8. **Virtualización**: Es una tecnologia que permite crear una versión virtual de un recurso fisico, como un servidor, sistema operativo o red.

9. **Comunicaciones**: Se refieren al proceso mediante el cual dos o mas dispositivos, sistemas o programas intercambian información.

10. **Servidor**: Es un sitema informático diseñado para proporcionar servicios, recursos o datos a otros dispositivos o programas.

11. **CPU**: Es el componente principal de un ordenador o sistema informático encargado de interpretar y ejecutar las instrucciones de los programas.

12. **Cifrado**: Es un proceso mediante el cual la información o datos se transforman utilizando algoritmos matematicos.

13. **Utilitarios**: Son aplicaciones diseñadas para ayudar a gestionar, mantener, optimizar y proteger lel sistema operativo y los recursos.

14. **Nube**: Se refiere a la entrega de servicios informáticos a travez de internet, en lugar de hacerlo desde un servidor o computadora local.

15. **Ciberseguridad**: Es el contenido de prácticas, tecnologias y procesos diseñados para proteger sistemas informáticos, redes y dispositivos.

 # Glosario semana 2

1. **CPU**: El CPU o Unidad Central de Procesamiento, es el componente principal
de un ordenador que ejecuta las instrucciones de los programas informáticos.
Es considerado el cerebro del sistema, ya que realiza las operaciones básicas
del cálculo, lógica y comunicación con los otros sistemas.

2. **Driver**: Es un software que permite que el sistema operativo y los programas
puedan comunicarse y controlar un hardware específico, como una
impresora, tarjeta gráfica, teclado, ratón, disco duro, entre otros.

3. **Firmware**: El firmware es un tipo de software de bajo nivel que reside en un
dispositivo como un ordenador, un smartphone u otro aparato electrónico.
Ayuda al sistema operativo (SO) a comunicarse con el hardware y proporciona
instrucciones para permitirle realizar tareas.

4. **GPU**: Es un componente de hardware especializado en el procesamiento y
renderizado de imágenes, videos y gráficos en un ordenador o dispositivo
electrónico.

5. **HDD**: Es un dispositivo de almacenamiento de datos que utiliza discos
magnéticos giratorios para guardar información. Con buena capacidad y costo
accesible, pero con menor velocidad y mayor fragilidad que los SSD.

6. **Núcleo**: Es la parte central de un sistema operativo que actúa como
intermediario entre el hardware (como la memoria, CPU y dispositivos) y el
software. Su función principal es gestionar los recursos del sistema, controlar
los procesos y permitir la ejecución de aplicaciones, asegurando un
funcionamiento seguro y eficiente del ordenador.

7. **Periférico**: Es un dispositivo externo que se conecta a un ordenador o sistema
para ampliar sus capacidades o permitir la interacción con el usuario. Los
periféricos no forman parte del hardware interno principal del equipo, pero son
esenciales para realizar tareas específicas.

8. **RAM**: (Memoria de Acceso Aleatorio, por sus siglas en inglés Random Access
Memory) es un tipo de memoria informática que se utiliza para almacenar
temporalmente los datos y programas que el procesador necesita mientras
está en funcionamiento.

9. **Sistema Operativo**: Su función principal es gestionar los recursos del
sistema, como la memoria, el procesador, los dispositivos de entrada/salida y
el almacenamiento, para que los programas puedan funcionar correctamente
y de manera eficiente.

10. **SSD**: Es un dispositivo de almacenamiento de datos que utiliza memoria flash
para guardar información mejorando significativamente el rendimiento general
del equipo, especialmente en tiempos de arranque, carga de aplicaciones y
transferencia de archivos.

# Glosario semana 3

1. **Antimalware**: Es el conjunto de herramientas y técnicas diseñadas para prevenir, detectar y eliminar software malicioso

2. **Arquitectura**: Se refiere al diseño y organización fundamental de los componentes de un sistema informático, incluyendo tanto el hardware como el software, y cómo estos interactúan para ejecutar tareas y procesar información.

3. **Memoria virtual**: La memoria virtual es una técnica utilizada por los sistemas operativos para ampliar la capacidad de la memoria RAM física disponible en un ordenador, permitiendo que el sistema utilice espacio en el disco duro como si fuera memoria adicional

4. **Scheduling**: Es el mecanismo que permite a un sistema operativo gestionar el acceso a la CPU y otros recursos, organizando la ejecución de procesos y hilos para optimizar el rendimiento, la equidad y la respuesta del sistema.

5. **Hilos**: Un hilo (o thread en inglés) es la unidad más pequeña de ejecución dentro de un proceso. Un proceso puede contener uno o varios hilos que se ejecutan concurrentemente, compartiendo recursos como memoria y archivos, pero cada hilo tiene su propio contador de programa, pila y registros.
